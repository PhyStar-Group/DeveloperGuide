入门手册
===================================
入门手册主要介绍如何快速的使用PhyStar。根据用户背景，我们将会分为两个部分

1. 第一部分 **新手指南：**
    - 主要针对没有linux开发与使用经验的同学，指南尽可能把一些关键的步骤列出。本部分学习后，可尝试开展第二部分的内容。
2. 第二部分 **快速开始：**
    - 针对具备一定Linux开发与使用经验的同学，能够快速了解"如何开始一个新算例"。

如果需要了解PhyStar的最新资讯，可以收藏网址： https://phystar-group.github.io/
